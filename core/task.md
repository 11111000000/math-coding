# Math Coding Core — v2

## Problem

Why does this packet exist as a packet, not just a file? Because
the convention requires it. The convention is **fractal**: it
applies to itself. So `core/core.md` (the convention) is wrapped
in a packet that demonstrates compliance with itself.

## Desired outcome

A packet that contains the canonical convention document
(`core.md`) and is itself a valid packet under the convention.

## Constraints

- The packet has all required files (packet.yaml, task.md,
  assumptions.yaml)
- The convention document (core.md) is grounded in the 8 theory
  documents
