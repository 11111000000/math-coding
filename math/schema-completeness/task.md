# schema-completeness

## Problem

The convention's schema is functional but incomplete. Five
small improvements make it more navigable for LLM agents
that load the convention into context.

## Desired outcome

Five additive changes (1-3 lines each) that improve
discoverability without breaking axiom Self-Application.

## Constraints

- Each change is **additive** (no breaking changes).
- The 16 self-tests still pass.
- axiom Self-Application: PROVEN.
