# 00-difference

## Thesis

A proposition and its implementation are distinct. The
proposition lives in language; the implementation lives in
executable symbols. Without this distinction, convention has
nothing to bridge.

## Antithesis

If proposition equals implementation, no convention is
needed. Code that explains itself has no decision to record.

## Synthesis

A0 grounds math-coding on difference. Each axiom that follows
exists because some proposition differs from some
implementation. Curry-Howard (A2) names this bridge.

## Surface impact

touches: convention's foundation [FROZEN]

## Proof

axiom A2 (Curry-Howard) instantiates the bridge. axiom A6
(self-application) verifies that the bridge closes.
