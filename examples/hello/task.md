# Example hello packet — sketch lifecycle (in examples/)

## Problem

A reader needs a minimal example of a packet to understand the
convention. Without a concrete example, the convention is
abstract and hard to grasp.

## Desired outcome

A sketch packet with all required files and minimal content,
serving as a template that readers can copy. The packet
demonstrates lifecycle=sketch, which requires only packet.yaml,
task.md, and assumptions.yaml.

## Constraints

- No formal model required (sketch allows no artifacts)
- Minimal content while still satisfying structural invariants
- All epistemic markers are well-formed (enum values)
