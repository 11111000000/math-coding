# Theory 05 — Assumption Set

## Problem

`assumptions.yaml` lists implicit assumptions but does not say
*how* they are used in the proof. Is an assumption an axiom
that can be trusted? A hypothesis that should be tested? A
judgment that cannot be challenged?

## Desired outcome

A document that:
- Treats the assumption set as $\Sigma$ in a logical theory
- Defines what it means to "derive" a postcondition from
  specification and assumptions
- Connects to the epistemic markers (fact, hypothesis, judgment,
  unknown): each has a different proof-theoretic role
- Gives a concrete example of $\Sigma \vdash \text{Spec} \models Post$

## Constraints

- The formal language is first-order logic over program state
- An example uses typical math-coding assumptions
- Reference to proof theory and Curry-Howard
